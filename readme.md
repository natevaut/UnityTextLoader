# Unity Text Loader

***Proof of concept***

----

Load and parse text content from file for display in Unity.

Page content is stored in XML files. Example:
```xml
<?xml version="1.0" encoding="UTF-8"?>
<page title="Lorem Ipsum">
    <element x="0" y="100" width="500" height="100">
        Lorem ipsum
    </element>
</page>
```
